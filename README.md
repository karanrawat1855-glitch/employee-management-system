# Employee Management System

## Tech Stack
- Backend: Spring Boot
- Frontend: React
- Database: MySQL

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employees

## How to Run

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm start  

## URLs
Frontend: http://localhost:3000  
Backend: http://localhost:8082
